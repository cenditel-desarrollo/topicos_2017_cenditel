Nueva estructura

En vista de las múltiples instancias requeridas para el sistema de consulta abierta, se planteo una nueva estructura para el sistema de consulta abierta, separandolo en 3 partes. Una independiente para la gestión de perfiles y modelado de tópicos, una para el gestor de consultas y la última para manejar la participación en las consultas generadas en el gestor.
